## NEW FILE: varieties.json

A brand new file with 13 coffee variety objects. Each has:
- id, name, aliases, origin_country, origin_region
- botanical_group, genetic_lineage
- flavor_archetype: { acidity, body, sweetness, fruity, earthy, complexity }  (all 1–10)
- flavor_notes: ["Jasmine", "Bergamot", ...]
- cup_profile_summary: short text description
- known_for: longer editorial text
- famous_examples: ["Hacienda La Esmeralda (Panama)", ...]
- best_countries: ["Panama", "Ethiopia"]
- altitude_preference_m, yield_potential, disease_resistance
- specialty_market: e.g. "Ultra-premium", "Mainstream Specialty"
- first_cup_of_excellence
